## Dependencies

- [Node.js](https://nodejs.org/en/) >= 8.0.0
- [Yarn](https://yarnpkg.com/pt-BR/docs/install)

## Getting started

1. Clone this repository;
2. `cd PaleMoon`;<br />
3. Run `yarn` to install dependencies.<br />
4. Run `yarn start`.
